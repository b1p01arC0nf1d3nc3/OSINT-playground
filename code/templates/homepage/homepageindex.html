About OSINTplayground

OSINTplayground is a learning tool to help beginners understand the fundamentals of performing open-source intelligence (OSINT) investigations. The different aspects of an investigation are broken down into a number of challenge rooms. Each challenge room tasks the user with practicing key skills and researching tools used by professionals.

Additionally, all sites can be navigated freely, which allows for more open investigation of example user activity.
